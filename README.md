# Gestion des Dépenses

Une application web simple pour suivre et gérer vos dépenses personnelles. Vous pouvez ajouter, modifier, supprimer des dépenses et filtrer par catégorie. Les données sont sauvegardées localement dans le navigateur grâce à `localStorage`.

---

## Fonctionnalités

- Ajouter une dépense avec :
  - **Titre**
  - **Montant**
  - **Catégorie** (Quotidien, Loisirs, Transport, Courses)
- Modifier une dépense existante
- Supprimer une dépense
- Filtrer les dépenses par catégorie
- Affichage du **total des dépenses**
- Sauvegarde automatique dans le navigateur via `localStorage`

---

## Technologies utilisées

- HTML5
- CSS
- JavaScript (Vanilla JS)

---

## Installation et utilisation

1. Cloner ce dépôt ou télécharger les fichiers.
2. Ouvrir `index.html` dans un navigateur moderne.
3. Ajouter, modifier ou supprimer vos dépenses directement dans l’interface.
4. Les données sont automatiquement sauvegardées dans le navigateur.
